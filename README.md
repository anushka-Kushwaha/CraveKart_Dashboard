# 🍽️ CraveKart Analytics Dashboard

## 📌 Overview

CraveKart Analytics is an interactive Power BI dashboard designed to analyze food ordering patterns using quantity-based data. The goal of this project is to uncover insights into customer preferences, high-performing food items, and transaction behavior.

## 🎯 Objective

To understand what is working well for the platform by analyzing:

* Customer ordering patterns
* Top-performing food items
* Category-wise demand
* Payment preferences
* Monthly transaction trends

## 🛠️ Tech Stack

* **Power BI Desktop** – Data visualization
* **Power Query** – Data cleaning and transformation
* **DAX (Data Analysis Expressions)** – Calculated measures and KPIs
* **Data Modeling** – Relationships across tables

## 📂 Data Source

* Dataset sourced from **Kaggle**
* Contains structured food delivery data including:

  * Order quantity
  * Food categories and items
  * Customer membership type (Gold, Regular)
  * Payment method
  * Order date and delivery status

## 📊 Dashboard Features

### 🔹 KPI Metrics

* **Total Transactions:** 3K
* **Total Quantity:** 15K
* **Average Quantity per Order:** 5.5

### 🔹 Key Visualizations

* Monthly quantity trend analysis
* Transactions by food category and member type
* Top ordered food items
* Payment method distribution
* Delivery status (Delivered vs Cancelled)
* Monthly transaction trend

## 📈 Key Insights

* **Main Course and Snacks dominate demand**
* **Samosa and Butter Chicken are top-performing items**
* **UPI is the most preferred payment method**
* **High delivery success rate (~94%) indicates strong operations**
* **Peak order volume observed in February and March**
* **Gold members contribute higher per-category volume, while Regular users dominate overall

## 📌 Business Understanding

This dashboard helps in:

* Identifying high-demand food items
* Understanding customer behavior patterns
* Analyzing payment preferences
* Monitoring operational efficiency



## 🚀 Outcome / Learnings

* Built an end-to-end Power BI dashboard from raw dataset
* Applied DAX for KPI creation and insights extraction
* Identified trends without revenue-based metrics
* Improved data storytelling and visualization skills
